Hexo for Linux 16.04

#### install hexo
```
> sudo apt install nodejs-legacy

> sudo apt install npm

> sudo npm install -g hexo
```
https://hexo.io/zh-cn/docs/index.html

#### prepare

```
>mkdir github.web
>cd github.web
>sudo hexo init
>sudo npm install
```
#### start server
```
>sudo hexo server
```
